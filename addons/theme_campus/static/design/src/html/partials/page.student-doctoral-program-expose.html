Expose - DBA in law

started on 2nd Dec 2019

Task description

The writing of an exposé is an important milestone in the doctoral study process. In the DBA doctoral program the exposé is a research paper about the students’ doctoral issue. The doctoral student can compose the proposal in his preferred language (that is the language the dissertation will be written in). This can be English or any other language. For the assessment of the exposé, it is mandatory that it is submitted in English. A notarized translation is permitted.
